## 获取通知
该命令可以自动获取 chrome的打印
```
dbus-monitor "interface='org.freedesktop.Notifications'"
```

手动发送

```
notify-send "1231312312312312312313131321, 15645645645,等等4啊5的4爱上5的454手动564"


```

只需要 带有 string 的通知
